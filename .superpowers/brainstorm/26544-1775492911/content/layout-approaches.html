A — Worktree Tab Row

A second tab strip beneath the project tabs. Each worktree is a tab (branch name). Switching tabs switches the active worktree — terminals, threads, and git context all update. A "+ New" button creates a worktree inline.

B — Worktree Sidebar Panel

A dedicated panel (between thread sidebar and center content) listing all worktrees with branch name and terminal status indicator. Clicking switches context. Compact — doesn't push the center panel.

C — Grouped Terminal Tabs

Terminals in the center panel tab bar are grouped under inline worktree labels. All worktrees' terminals are visible at once; click any to jump directly. No worktree switcher — the tab bar IS the worktree manager.
